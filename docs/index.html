How It Works

Three steps. Zero friction.

SecretShields works at the clipboard level, so it protects every app you paste into.

1

Copy as usual

Copy text containing an API key, database URL, or any credential. SecretShields detects it within ~1 second.

2

Clipboard is masked

Your clipboard is silently rewritten with a masked version. A status bar pulse confirms the action — no workflow interruption.

3

Paste anywhere safely

Paste into Cursor, Copilot, Claude Code, terminals, Slack, or any app. The secret never leaves your machine.

Features

Built for developers who move fast

⚡

Clipboard-First Masking

Masks secrets at the system clipboard level within ~1s of copying. Protects AI chats, terminals, pastebins, and any other paste target.

🔄

Restore with TTL

Need the real value? Restore for a configurable window (default 60s). SecretShields automatically starts a rotation reminder countdown.

🔔

Rotation Reminders

When you expose a secret, a countdown fires. On expiry, you get a notification with a direct link to the provider's rotation page.

📋

Exposure Log

Track which secrets were exposed, when, and their rotation status. Only masked previews are stored — never raw values.

📋

Editor Paste Widget

Optionally intercepts paste in code editors with a "Paste with SecretShields masking" option. Configurable: offer, auto, or off.

🛡

Always-Visible Signal

Status bar pulse and optional Output channel logging ensure you always know when masking occurs — even if notifications are suppressed.
